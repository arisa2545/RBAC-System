# 🛡️ Role-Based Access Control (RBAC) System

## 🚀 How to run project (Getting Started)

ระบบนี้ใช้ Docker ในการจำลองสภาพแวดล้อมทั้งหมด สามารถรันโปรเจกต์ได้ง่ายๆ ตามขั้นตอนดังนี้:

### 1. สิ่งที่ต้องมีในเครื่อง (Prerequisites)
* Docker ติดตั้งและเปิดใช้งานอยู่
* Git

### 2. ขั้นตอนการติดตั้ง
1. **Clone Repository:**
   ```bash
   git clone <ใส่-URL-ของ-Repo-ที่นี่>
   cd <ชื่อโฟลเดอร์โปรเจกต์>

2. **Start service with Docker:**
   ```bash
   docker-compose up -d

3. **Access web with URL:**
   ```bash
   http://localhost:5010

## 💻 Tech Stack

**Frontend:**
* React (Vite)
* TanStack Router (Routing)
* TanStack Query (Data Fetching & State Management)
* Tailwind CSS & shadcn/ui (UI Components)
* React Hook Form (Form Handling)

**Backend:**
* NestJS (Node.js Framework)
* Prisma (ORM)
* JSON Web Token (JWT Authentication)

**Database & Infrastructure:**
* MySQL
* Docker & Docker Compose
